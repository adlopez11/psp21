Tarea 6 PSP 2.1

DESCARGA

Para descargar el proyecto desde el repositorio de GitHub, seguir los siguientes pasos:

- Abrir en una consola o shell del sistema
- Ubicarse en la raiz de la carpeta de proyectos
- Ejecutar el siguiente comando:

git clone https://github.com/adlopez11/psp21.git proyectopsp2

Se creará una carpeta llamada "proyectopsp21" con todo el contenido del repositorio, se puede cambiar el nombre de la carpeta.


EJECUCIÓN

Luego de compilar el proyecto, ubicados en la raiz de la carpeta, la aplicación se puede ejecutar con el comando:

java -cp target/psp2.1-1.0-SNAPSHOT.jar edu.uniandes.ecos.ui.Main

La aplicacion solicitará los valores de p y dof
